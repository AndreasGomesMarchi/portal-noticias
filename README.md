# portal-noticias
Esse projeto foi feito para a atividade 04 da matéria de Programação web - Quinto semestre

📰 Portal de Notícias (React + TypeScript)

⚙️ Como Rodar o Projeto

1. Clonar o repositório
git clone https://github.com/AndreasGomesMarchi/portal-noticias.git
cd portal-noticias

2. Instalar dependências
npm install

3. Rodar o projeto
npm run dev

4. Acessar no navegador
http://localhost:5173
